## üéØ Common Workflows

### Workflow 1: Generate Posts from Portfolio

```
1. Login with Facebook Business account
2. Navigate to "Portfolios" section
3. Click "Upload Portfolio"
4. Select PDF or Word document with business info
5. Wait for upload to complete
6. Click "Generate Posts" button
7. AI generates 3 engaging posts automatically
8. Posts appear in "Pending Review" queue
9. Review each post, edit if needed
10. Click "Approve" to add to queue
11. Schedule or post immediately
```

### Workflow 2: Schedule Weekly Posts

```
1. Go to Posts section
2. Create 7 posts for the week
3. For each post:
   - Add media from library
   - Add hashtags
   - Click "Schedule"
   - Set date/time (e.g., Monday 9 AM)
4. All posts will auto-publish at scheduled times
5. Track engagement on Dashboard
6. Analyze performance in Analytics
```

### Workflow 3: Repurpose Media Across Posts

```
1. Upload hero image to Media Library
2. Create Post #1 with image + copy A
3. Create Post #2 with same image + copy B
4. Create Post #3 with same image + copy C
5. Schedule all three for different times
6. Same media, different messaging
7. Compare engagement to find best-performing copy
```

### Workflow 4: AI Post Improvement

```
1. Create manual post with initial text
2. Review in dashboard
3. Edit post, request AI suggestions
4. AI improves language and engagement
5. Compare original vs improved version
6. Choose best version
7. Add recommended hashtags
8. Publish or schedule
```

## üõ†Ô∏è Troubleshooting

### Posts Not Publishing

**Problem**: Scheduled posts don't publish
**Solution**: 
- Check Facebook page is selected
- Verify access token is valid
- Check scheduled time is in future
- Review error logs for details

### AI Posts Not Generating

**Problem**: Portfolio posts not created
**Solution**:
- Ensure text was extracted from portfolio
- Check Claude API key is valid
- Verify API has sufficient credits
- Try uploading different document

### Media Not Showing

**Problem**: Uploaded images don't appear
**Solution**:
- Check file format is supported (JPG, PNG, MP4)
- Verify file size < 100MB
- Refresh page after upload
- Check browser console for errors

### Performance Issues

**Problem**: Dashboard loads slowly
**Solution**:
- Delete old analytics data
- Archive processed portfolios
- Clear browser cache
- Use PostgreSQL instead of SQLite
- Close other applications
